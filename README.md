## 🏛️ The Mission
AI is generating code faster than humans can understand it. Architecture is rotting silently. Technical debt is compounding invisibly. **DevBrand is the antidote.**

We provide **AI-powered engineering judgment** for modern repositories. We don't just summarize code; we judge its long-term impact, detect "AI slop," and predict future architectural chaos before it destroys your velocity.

---

## 🚀 The Verdict Engine (Core Intelligence)
The heart of DevBrand is a surgical judgment engine that cross-references static metrics, dependency graph disruption, and semantic intent.

### 🧠 Judgment Categories
- **Maintainability Verdict**: Predicting future pain. Is complexity accelerating faster than test coverage?
- **AI Slop Detection**: Identifying repetitive patterns, fake abstractions, and hallucinated architecture common in AI-generated bloat.
- **Architecture Verdict**: Real-time analysis of service coupling, circular dependencies, and infrastructure overengineering.
- **Engineering Culture Verdict**: Who built this? Identifying personas from *Ex-FAANG Reliability Engineer* to *Startup Speed Demon*.

---

## 🗺️ System Topology (The Living X-Ray)
DevBrand provides a real-time, interactive distributed systems map that visualizes the "Living State" of your infrastructure.
- **Flow Visualization**: See data moving through the event mesh.
- **Resilience Mapping**: Inspect AI model fallbacks and retry semantics.
- **Debt Hotspots**: Automatic highlighting of architectural smells directly on the map.

---

## 📂 Monorepo Architecture
DevBrand is engineered as a high-reliability distributed monorepo.
- **`apps/web`**: High-fidelity dashboard & Repo Roast interface.
- **`packages/repo-intelligence`**: The multi-layer analysis core (The Verdict Engine).
- **`packages/ai-sdk`**: Adaptive inference router with automatic failover.
- **`modules/core`**: Resilient event mesh and workflow orchestration.

## 🛠️ Implementation Guide

### Prerequisites
- Node.js `^22.0.0`
- Docker & Docker Compose
- Pnpm `^9.0.0`

### Quick Start
```bash
# Clone the repository
git clone https://github.com/bansalbhunesh/devbrand.git

# Install dependencies
pnpm install

# Initialize infrastructure
docker-compose up -d

# Launch the Intelligence Layer
pnpm dev
```
